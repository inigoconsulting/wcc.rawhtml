wcc.rawhtml Installation
------------------------

To install wcc.rawhtml using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.rawhtml`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.rawhtml

* Re-run buildout, e.g. with:

    $ ./bin/buildout
